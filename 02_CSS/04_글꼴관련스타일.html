글꼴 테스트 Font Style Test

font-size : 글꼴의 크기를 변경할 때 사용

선택자{ font-size:크기(px|em|%) }
  • 고정크기 px테스트
  • 가변크기 em테스트
  • 가변크기 %테스트

font-weight : 글꼴의 굵기를 표현할 때 사용

  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-style : 텍스트 문구를 기울이고자 할 때 사용

선택자{ font-style:normal|itailc|oblique; }

italic 기울임 글꼴

oblique 원래 글자를 기울인셈
